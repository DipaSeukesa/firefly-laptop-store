# Firefly Laptop Store

Aplikasi web modern untuk showcase laptop dengan desain yang menarik dan komponen yang terorganisir dengan baik.

## Teknologi

- **React** - Framework UI
- **Vite** - Build tool yang cepat
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Icon library

## Struktur Komponen

- `Navbar` - Navigasi header
- `Hero` - Section hero dengan search bar
- `Filter` - Filter kategori produk
- `ProductCard` - Card untuk menampilkan produk
- `ProductDetailModal` - Modal untuk detail produk
- `Footer` - Footer aplikasi

## Instalasi

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```

3. Build untuk production:
```bash
npm run build
```

## Fitur

- ✅ Pencarian produk
- ✅ Filter berdasarkan kategori
- ✅ Detail produk dengan modal
- ✅ Desain responsif
- ✅ Animasi firefly background
- ✅ UI modern dengan Tailwind CSS
